# PostgresSQL in Linux environment

![alt text](https://linagora.com/wp-content/uploads/2018/06/PostgreSQL-logo-1-1920x1080.png)

### Use the PostgresSQL command line

```sudo -i -u postgres```

#### access remote db

```psql -h <hostname> -p <port #> -U <username>```

#### list the databases
```\l```

#### quit the prompt
```\q```
